📋 Campaigns vs Cadences — When to Use Which

          

          

            

              

                
📢 Campaigns

                
A campaign is a time-bound, goal-oriented marketing initiative that coordinates multiple channels and content types around a single objective.

                
    
                  
  • Has a defined start and end date
    • 
                  
  • Targets a specific persona or segment
    • 
                  
  • Includes multi-channel content (email, social, ads)
    • 
                  
  • Tracks budget and ROI
    • 
                  
  • Use when: launching a product, seasonal push, event promotion, B2B outreach blitz
    • 
                

              

              

                
📧 Cadences

                
A cadence is an automated sequence of emails/SMS triggered when a contact enters a specific stage or meets certain criteria.

                
    
                  
  • Evergreen — runs continuously once activated
    • 
                  
  • Triggered by contact actions (signup, trial start, etc.)
    • 
                  
  • Sequential steps with defined delays
    • 
                  
  • Personalized per contact
    • 
                  
  • Use when: nurturing leads, onboarding, trial-to-paid conversion, re-engagement, win-back
    •

🔑 Meta Token Setup — Facebook & Instagram Publishing

              Click to expand step-by-step guide
            

            

              
To activate Facebook and Instagram posting, you need 3 values added to Vercel. Here's exactly how to get them:


              


                

                  
Step 1 — Create a Meta App

                  
    
                    
  1. Go to developers.facebook.com/apps
    2. 
                    
  2. Click Create App → choose "Business" type
    3. 
                    
  3. Name it "SmartSwing Publisher" → select your Business account
    4. 
                    
  4. Add product: Facebook Login + Instagram Graph API
    5. 
                  

                


                

                  
Step 2 — Get a Never-Expiring Page Access Token

                  
    
                    
  1. Go to Business Settings → System Users
    2. 
                    
  2. Click Add → name it "SmartSwing Publisher" → role: Admin
    3. 
                    
  3. Click Add Assets → select your Facebook Page → toggle Full Control
    4. 
                    
  4. Click Generate New Token → select your App → set expiry to Never
    5. 
                    
  5. Enable permissions: pages_manage_posts, pages_read_engagement, instagram_basic, instagram_content_publish
    6. 
                    
  6. Copy the token → this is your META_PAGE_ACCESS_TOKEN
    7. 
                  

                


                

                  
Step 3 — Find Your Page ID & Instagram Account ID

                  
    
                    
  1. Page ID: Go to your Facebook Page → click About → scroll to bottom for Page ID, OR visit graph.facebook.com/me/accounts?access_token=YOUR_TOKEN
    2. 
                    
  2. Instagram Account ID: In Graph API Explorer, call /me/accounts → find the page → call /{page-id}?fields=instagram_business_account → the id field is your META_IG_ACCOUNT_ID
    3. 
                  

                


                

                  
Step 4 — Add to Vercel Environment Variables

                  
Go to Vercel → SmartSwing AI → Settings → Environment Variables and add:

                  

                    META_PAGE_ACCESS_TOKEN = YOUR_NEVER_EXPIRING_TOKEN

                    META_PAGE_ID = YOUR_FACEBOOK_PAGE_ID

                    META_IG_ACCOUNT_ID = YOUR_INSTAGRAM_ACCOUNT_ID
                  

                  
After adding, click Save then Redeploy the project for the variables to take effect.

                


                

                  💡 Graph API Explorer: Use developers.facebook.com/tools/explorer to test your token and find IDs before adding to Vercel.
